# Appendix
  
#### Colors:  (For StringColor only)
0 -> White

1 -> Red

2 -> Bright Green

3 -> Blue

4 -> Gold

5 -> Grey

6 -> Black

7 -> Tan

8 -> Orange

9 -> Yellow

10 -> Dark Green

11 -> Purple

#### DrawModes: (Used by boxes only)
0 -> 25% opacity

1 -> 50% opacity

2 -> 75% opacity

3 -> Color blend 1 (Transparent + Highlight)

4 -> Color Blend 2 (Makes image pure black)

5 -> Normal Draw

6 -> Color Blend 3 (Screen + Transparent)

7 -> Highlight
